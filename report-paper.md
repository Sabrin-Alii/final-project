Traffic Prediction System Report

1. Introduction

Traffic congestion is a major problem in modern cities. This project aims to predict traffic levels using machine learning techniques.

2. Objective

The main objective is to develop a system that can classify traffic into:

- Low
- Medium
- High

3. Methodology

- Data preprocessing
- Feature selection
- Model training using machine learning
- Deployment using FastAPI and React

4. System Design

The system consists of:

- Frontend (React)
- Backend (FastAPI)
- Machine Learning Model

5. Results

The system successfully predicts traffic levels based on user input.

6. Conclusion

The project shows how AI can be used in real-world applications to improve decision-making.
